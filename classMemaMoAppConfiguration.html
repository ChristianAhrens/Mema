Detailed Description

XML-backed configuration manager for the Mema.Mo monitor application.

Persists the user's chosen Mema service (host/port), output visualization type, metering colour, and look-and-feel selection across application launches. Inherits the dump/watch/version-conflict infrastructure from AppConfigurationBase.

Note
Part of the Mema tool suite. Mema.Mo is the monitoring companion to the Mema audio-matrix server: it receives streamed audio buffers over TCP and renders them as level meters, 2-D spatial field, waveform, or spectrum. This configuration class stores everything Mema.Mo needs to reconnect and restore its last visual state.
